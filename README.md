## 🚀 升级指南

由于小说目录默认储存在插件目录，所以升级后会不同步旧版本的小说。

同步旧版本小说步骤：

1. `F1` 运行 `my-reader:打开本地文件目录`
2. 找到 `**\.vscode-insiders\extensions` 目录下 `kennyou.my-reader-${版本号}\book`
3. 复制对应版本的文件到当前版本的小说目录里面

## 🎉 功能

- [x] 支持格式:txt、epub
- [x] 支持起点网小说搜索和阅读
- [x] 支持笔趣阁小说搜索和阅读
- [x] 支持侧边栏章节显示(epub、在线)
- [x] 阅读页面菜单支持上下一章
- [x] 自定义阅读界面的 CSS 和 HTML
- [x] 状态栏进度显示
